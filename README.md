# Alphavantage
React + TypeScript + TailwindCSS
## Running application
Update files
Place API Key in src/constants.ts file
Use node version 18+

cd stock-dashboard
npm i
npm start
Then open http://localhost:3000/.

## Endpoint used
TIME_SERIES_DAILY - https://www.alphavantage.co/query?function=TIME_SERIES_DAILY&symbol=IBM&apikey=demo
